# 🌟 Profile Card

A simple and clean Profile Card built using **HTML** and **CSS**.  
It displays basic user information like name and profile picture in a visually appealing card layout.

---

## ✨ Features

- Clean and minimal design  
- Profile image display  
- User name and basic info  
- Responsive layout for all devices  

---

## 🛠️ Technologies Used

- HTML  
- CSS  

---

## 🎯 Purpose

The purpose of this project is to practice front-end development skills and create a **reusable profile card component** for portfolios or personal websites.
